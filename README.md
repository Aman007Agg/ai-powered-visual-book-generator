# AI Powered Visual Book Generator

## Overview

AI Powered Visual Book Generator is a browser-based application that transforms raw book content (text and images) into visually designed book pages using Large Language Models.

This project is being developed as part of the EY GenAI Engineering Assignment.

## Tech Stack

### Frontend
- Angular 20
- Angular Material
- SCSS
- TypeScript

### Backend
- FastAPI
- Python

### AI
- OpenAI GPT
- Structured JSON Outputs
